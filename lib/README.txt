All files in this folder are developed in FMUP namespace.

This namespace purpose is to extends FMU framework capacities to evolve in namespace architecture without changing
actual working for FMU basics.

Second point of this framework is to allow projects to extends FMU/FMUP framework in their actual use-cases.

The vision of this project is NOT to recode a FULL Framework, but intends to work along with other existing sources.
